# Веб-приложение для учета времени сотрудников

Учебный проект для практики. Реализован на JavaScript + React. Проект сделан максимально простым, но теперь логика разнесена по отдельным файлам-модулям. Главный файл `src/main.jsx` связывает модули между собой, стили находятся в `src/App.css`.

## Что реализовано

- модуль сотрудников;
- модуль учета рабочего времени;
- модуль отчетности;
- интеграция модулей между собой через общий React state и localStorage;
- добавление, редактирование и удаление сотрудников;
- добавление и удаление записей рабочего времени;
- автоматический расчет часов и зарплаты;
- просмотр отчета прямо в веб-приложении;
- экспорт отчета в Excel `.xlsx`;
- сохранение данных в браузере после перезагрузки страницы;

## Как запустить

1. Установить Node.js с официального сайта.
2. Распаковать архив с проектом.
3. Открыть папку проекта в терминале.
4. Выполнить команду:

```bash
npm install
```

5. Запустить проект:

```bash
npm run dev
```

6. Открыть адрес, который появится в терминале, обычно:

```text
http://localhost:5173
```

## Как собрать проект

```bash
npm run build
```

После сборки файлы появятся в папке `dist`.

## Структура файлов

- `src/main.jsx` — главный файл приложения, связывает все модули;
- `src/modules/EmployeesModule.jsx` — модуль сотрудников;
- `src/modules/TimeModule.jsx` — модуль учета рабочего времени;
- `src/modules/ReportsModule.jsx` — модуль просмотра отчета;
- `src/modules/reportStorage.js` — экспорт отчета в Excel;
- `src/modules/DocumentationModule.jsx` — краткая документация внутри приложения;
- `src/utils/storage.js` — сохранение и чтение данных из localStorage;
- `src/utils/timeUtils.js` — расчет количества часов;
- `src/data/demoData.js` — начальные демо-данные.

## Описание модулей

### 1. Модуль сотрудников
Хранит список сотрудников: ФИО, должность, ставка в час.

### 2. Модуль учета времени
Позволяет выбрать сотрудника, указать дату, время начала, время окончания и комментарий. После добавления записи автоматически считается количество часов.

### 3. Модуль отчетности
Использует данные из двух предыдущих модулей и показывает сводку: общее количество часов, ставка, сумма к оплате. Отчет можно экспортировать в Excel.

## Интеграция модулей

Данные передаются между модулями через общий state React-компонента `App` в файле `src/main.jsx`. Формат данных — JSON-объекты. Для сохранения используется `localStorage`.

Пример объекта сотрудника:

```json
{
  "id": "emp_1",
  "name": "Иванов Иван",
  "position": "Программист",
  "rate": 300
}
```

Пример записи времени:

```json
{
  "id": "time_1",
  "employeeId": "emp_1",
  "date": "2026-06-05",
  "start": "09:00",
  "end": "17:00",
  "comment": "Работа над модулем отчетности"
}
```
